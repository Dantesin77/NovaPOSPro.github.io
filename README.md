# Página Web - Nova POS Pro

## Archivos incluidos:
- `index.html` - Página web completa y lista para usar

## Cómo usar:

### Opción 1: Subir a un hosting (Recomendado)
1. Sube los archivos a un hosting:
   - **Netlify** (gratis): Arrastra la carpeta al panel de Netlify
   - **Vercel** (gratis): Conecta tu GitHub
   - **Hosting propio**: Sube via FTP

### Opción 2: Enviar como archivo
1. Comprime la carpeta `pagina-web` como .zip
2. Envía el archivo al cliente

### Opción 3: Abrir localmente
1. Abre `index.html` directamente en el navegador
2. Funciona sin internet (excepto las fuentes)

## Personalización:
Edita `index.html` para cambiar:
- Precios (líneas 157-158)
- Texto de contacto (al final del archivo)
- Números de WhatsApp y email

## Vista previa del programa:
Si tienes una captura de pantalla del programa, renómbrala a `preview.png` y colócala en la misma carpeta para que aparezca en la web.
