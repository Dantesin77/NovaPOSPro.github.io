Preguntas Frecuentes

¿Necesito internet para usar el programa?

No, el programa funciona 100% offline. No requiere conexión a internet.

¿Qué pasa cuando se vence la licencia?

El programa se bloquea y no permitirá hacer ventas hasta que renueves tu licencia.

¿Puedo usar lector de códigos de barras?

Sí, cualquier lector USB básico funciona. El lector envía los números como teclado y el producto se agrega automáticamente.

¿El programa maneja dólares y bolívares?

Sí, puedes cobrar en ambas monedas. El sistema convierte automáticamente según la tasa del día.

¿Puedo hacer respaldo de mis datos?

Sí, el programa tiene función de backup y restore para que nunca pierdas tu información.

¿Cómo recibo soporte?

Puedes contactarnos por WhatsApp o email. Respondemos lo más pronto posible.
